AI API Projesi

Görsel Tanıma ve Metin Analizi Uygulaması

Proje Hakkında

Bu proje, OpenAI ve HuggingFace gibi güçlü yapay zeka API'lerini kullanarak görsel tanıma ve metin analizi yeteneklerine sahip bir web uygulamasıdır. Node.js tabanlı bir altyapı üzerinde çalışmaktadır.

Özellikler

  • Yüklenen görsellerdeki nesneleri tanıma
  • Metin içeriklerini analiz etme ve özetleme
  • Duygu analizi yapabilme
  • RESTful API ile kolay entegrasyon

Kullanılan Teknolojiler

Node.js
JavaScript
OpenAI
Hugging Face
