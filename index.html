🧠 AI API Center – Unified Agent Platform

A modular, extensible backend platform for interacting with multiple AI providers through a single, unified interface. It's designed to abstract the complexities of various LLM providers, offering a consistent API for text generation and conversational chat.

🔑 Key Features

⚡ Multi-Provider Support
📡 Streaming Responses (SSE)
💬 Conversational Chat
📝 Prompt Templating
🔐 Authentication & Security
📊 Request Logging

🛠️ Tech Stack

Python (FastAPI)
PostgreSQL (SQLAlchemy)
JWT & Passlib
OpenAI / Google / Anthropic / DeepSeek
Jinja2

🏗️ Project Structure

unified-ai-api/
├── app/                     # Main FastAPI application
│   ├── auth/                # Authentication (JWT, passwords, middleware)
│   ├── DB_connection/       # Database managers (clients, chats, API keys, etc.)
│   ├── handlers/            # Handlers for each AI provider
│   ├── models/              # Pydantic and SQLAlchemy models
│   ├── routers/             # Request dispatcher
│   ├── utils/               # Utility modules (e.g., logger)
│   ├── root_prompt.txt      # Optional root prompt template
│   └── server.py            # FastAPI server entrypoint
├── tests/                   # Unit and integration tests
├── requirements.txt         # Python dependencies
└── README.md
